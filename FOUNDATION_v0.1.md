# FCStudio – FOUNDATION v0.1

## Vision
FCStudio est un SDK de reverse engineering des formats Frostbite. Le projet privilégie la compréhension des formats, la documentation et une architecture modulaire avant le développement de fonctionnalités avancées.

## Principes
- Comprendre avant de coder.
- Distinguer les faits des hypothèses.
- Documenter chaque découverte.
- Une responsabilité par classe.
- SDK indépendant des interfaces CLI.

## Réalisations Foundation
- Architecture du projet.
- Documentation de base.
- BinaryFile.
- Header Analyzer.
- String Analyzer.
- Entropy Analyzer.
- Pattern Scanner.
- Binary Comparator.
- Premiers lecteurs TOC/CAS.
